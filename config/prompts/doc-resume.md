## Résumé de document

Réalise un résumé du document joint.

**Paramètres** — si non précisés, demande-les en une seule fois :
- Longueur : express (5-10 bullets) | standard (1 page) | détaillé (plusieurs pages)
- Format : bullet points | paragraphes rédigés | structure avec titres et sous-sections
- Langue de sortie : (même que le document par défaut)
- Focus : tout le document | sections ou thématiques spécifiques : [préciser]

Quand les paramètres sont clairs, applique ce plan :

1. **Lis l'intégralité du document** et cartographie sa structure globale (parties, sections, thèmes principaux)
2. **Extrais l'idée principale** de chaque section ainsi que les informations clés qui la soutiennent
3. **Évalue le poids relatif** de chaque partie pour prioriser ce qui mérite d'apparaître dans le résumé
4. **Rédige le résumé** en respectant strictement la longueur et le format demandés
5. **Vérifie l'exhaustivité** : aucun point essentiel ne doit manquer, aucun élément hors document ne doit apparaître
6. **Cite les sources** entre parenthèses (page, section) pour chaque point important
